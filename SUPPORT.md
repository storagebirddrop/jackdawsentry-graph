# Support

## General Support

Use [GitHub Issues](https://github.com/storagebirddrop/jackdawsentry-graph/issues)
in this repository for:

- bug reports
- feature requests
- questions about setup, runtime behavior, or investigation workflows

When opening an issue, include the commit or release you are using, what you
expected to happen, what actually happened, and any relevant logs or screenshots.

## Maintainer Contact

For private coordination that does not belong in a public issue, contact:

- `jackdawsentry.support@dawgus.com`

## Security Issues

Do not open public issues for vulnerabilities.

Follow [SECURITY.md](SECURITY.md) and report security concerns privately to the
maintainers first.

## Scope

This support policy covers the standalone `jackdawsentry-graph` repository only.
It does not cover the private `jackdawsentry` platform, compliance dashboard, or
private operational workflows.
